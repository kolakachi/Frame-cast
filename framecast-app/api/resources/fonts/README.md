Framecast caption export fonts
================================

Fonts in this directory are copied into `/usr/local/share/fonts/framecast`
during the API/worker Docker build so FFmpeg/libass can render captions with
the same family names selected in the editor.

The bundled `.ttf` files mirror the Google Fonts exposed by the editor font
picker. They are Apache License 2.0 / SIL Open Font License fonts sourced from
Fontsource/Google Fonts.
