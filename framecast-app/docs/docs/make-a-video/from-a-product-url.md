# Start from a product URL

WyvStudio can read a product page or article and ground the whole plan in its actual content — ideal for DTC ad creative.

## 1. Open New Video → From URL / Article

Click **+ New Video**, then choose the **From URL / Article** source (or paste the URL directly into a One-shot prompt).

:::info 📸 Screenshot
`img/howto/url-source-pick.png` — the wizard source picker with "From URL / Article" highlighted.
:::

## 2. Paste the link

Paste the product or article URL. WyvStudio extracts the key copy, claims, and product details.

:::info 📸 Screenshot
`img/howto/url-paste.png` — the URL input with a product link pasted in.
:::

## 3. Set format and visuals

Choose aspect ratio, scene count, and visual source (AI images suit product ads; stock works for lifestyle). Add a reference character if you want a consistent presenter.

## 4. Review the plan and generate

WyvStudio writes scene scripts and image prompts grounded in the page. Tweak the lines, then **Generate** — same as [Create your first video](/getting-started/create-your-first-video) from step 4.

:::tip For ad testing
After generating, use [hook variants](/variants-and-hooks/generate-hook-variants) to spin up multiple openers from the same product, then [export every aspect ratio](/export-and-publish/export-every-aspect-ratio) in one pass.
:::
