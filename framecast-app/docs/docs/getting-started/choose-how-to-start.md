# Choose how to start

When you click **+ New Video**, you pick one of three starting points. They all land in the same editor — they just differ in how much WyvStudio does up front.

:::info 📸 Screenshot
`img/howto/new-video-three-cards.png` — the "How do you want to start?" dialog showing all three cards.
:::

## Start from Scratch
Build every scene yourself. Pick a title and format, then drop straight into the editor. **Choose this when** you already know exactly what you want, scene by scene.

## Generate from a brief
Tell WyvStudio your topic and it writes the script, breaks it into scenes, and pre-fills visuals. **Choose this when** you have a topic and want a structured multi-scene draft to refine.

## One-shot from a prompt
Type one line and get back a finished video — AI visuals, animation, voice, and music. **Choose this when** you want the fastest path from idea to a watchable draft. See [Create your first video](./create-your-first-video).

:::tip Which should I use?
Most people start with **One-shot** to get a draft fast, then use **Scratch** later once they know the exact structure they want to repeat.
:::
