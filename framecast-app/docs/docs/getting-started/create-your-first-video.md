# Create your first video

The fastest way to see what WyvStudio does: describe a video in one sentence and let **One-shot** plan the whole thing — script, scenes, voice, visuals, motion, captions, and music.

**Time:** ~3 minutes of your attention · **Credits:** varies by length and visual choices (shown before you generate).

## 1. Open the New Video dialog

From the dashboard, click **+ New Video** (top right).

:::info 📸 Screenshot
`img/howto/first-video-new-button.png` — the dashboard with the **+ New Video** button highlighted (top right).
:::

## 2. Choose One-shot

Pick **One-shot from a prompt** — the fastest path. (See [Choose how to start](./choose-how-to-start) for the other two options.)

:::info 📸 Screenshot
`img/howto/first-video-new-modal.png` — the "How do you want to start?" dialog with the three cards (Start from Scratch / Generate from a brief / One-shot from a prompt).
:::

## 3. Describe what you want

Type one or two sentences describing the video. Be concrete about the subject, tone, and any call to action — for example:

> A founder explaining how our supplement helped customers feel more focused, ending with a call to visit the site.

Use the pills under the box to set **visual source** (AI images / stock / audiogram), **aspect ratio**, **scene count**, and whether it's a **spokesperson** video. Add a reference **character** if you have one.

:::info 📸 Screenshot
`img/howto/first-video-prompt.png` — the One-shot content step: prompt typed in, control pills visible, a character selected.
:::

## 4. Review the plan

WyvStudio writes the full plan — each scene's script line plus its image/motion prompt. Tweak any line, toggle **captions** and **background music**, then check the estimated credit cost.

:::info 📸 Screenshot
`img/howto/first-video-plan.png` — the "Here's the plan" screen showing scene cards, the captions/music toggles, and the credit estimate.
:::

## 5. Generate

Click **Generate**. You'll see live progress as WyvStudio writes the voice, creates visuals, adds motion, and composes music.

:::info 📸 Screenshot
`img/howto/first-video-generating.png` — the generation progress screen with the step checklist.
:::

## 6. Refine in the editor

When it finishes you land in the **editor**. Play it back, then fine-tune any scene — swap a visual, change the voice, edit captions. When you're happy, head to [Export](/export-and-publish/export-every-aspect-ratio).

:::info 📸 Screenshot
`img/howto/first-video-editor.png` — the editor with the finished video in the preview.
:::
