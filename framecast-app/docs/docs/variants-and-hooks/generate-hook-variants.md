# Generate hook variants

The hook — your first line — decides whether anyone keeps watching. WyvStudio can write several hook options for your video and **score** each one, so you can test the strongest openers.

## 1. Open hook options

In the editor's **Config** panel, scroll to **Hook options**.

:::info 📸 Screenshot
`img/howto/hooks-block.png` — the Config panel's "Hook options" section.
:::

## 2. Generate (or regenerate)

Click **Generate hooks** (or **↻ Regenerate** if you already have some). WyvStudio writes 3–10 fresh hooks from your script and scores each 0–100 on pattern interrupt, specificity, curiosity, and emotional pull.

:::info 📸 Screenshot
`img/howto/hooks-generated.png` — the hook list sorted by score, each with a score badge and a one-line reason.
:::

:::tip No charge
Generating and scoring hooks is LLM text — it's part of the included tier, so it costs **0 credits**. Regenerate as many times as you like.
:::

## 3. Use a hook

Click **Use Hook** on the one you want — it replaces your scene-1 line.

## 4. Spin variants from your hooks

To test several at once, open **Variants** (top of the editor), turn on **vary by hook**, pick how many, and generate. Each variant reuses your existing visuals and only re-records the changed line — so a batch of hook variants is cheap, not a full re-render.

:::info 📸 Screenshot
`img/howto/variants-hooks.png` — the Variants page with "vary by hook" enabled and a hook count selected.
:::

:::note Spokesperson note
For lip-synced **spokesperson** scenes, a new hook changes the audio but not the talking video yet — so the hook-variant flow is best on AI-image scenes with voiceover.
:::
