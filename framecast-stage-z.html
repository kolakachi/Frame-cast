Key rule: A user can ignore every tool and still create videos exactly as they do today. The tools are available — not mandatory. Any tool's "→ Create Video" CTA opens the wizard with whatever context was gathered. Nothing is lost if they skip the tools.
What gets pre-filled into the wizard
Entry point Pre-filled into wizard User still chooses
Direct (dashboard) Nothing Everything
🔍 Niche Intelligence Niche, tone, visual style, format, audience profile Source content, voice, hook
📊 Competitor Research Tone, visual style, hook pattern, audience profile, length Source content, specific topic

📺 Series → New Episode All series defaults: voice, style, caption preset, music, aspect ratio, channel Episode content only
Four entry points — same workflow underneath
🧭
Start from Scratch
Niche browser → competitor research → hook lab → generate. Best for new creators or new niches.
🎯
I Know My Audience
Skip research. Free-text audience description → hook lab → generate. Best for established creators.
📄
I Have Content Ready
Paste script / URL / upload audio. No niche, no research required. Direct to generation.
🔁
New Episode
Pick a series. All defaults inherited. Provide content only. Calendar pre-suggests next slot.
The Loop
1
Find niche / pick audience
2
Research what's working
3
Pick hook → generate video
4
Export + schedule + post
5
Track → feed into next video
Any step can be skipped. The loop always ends at creation.
